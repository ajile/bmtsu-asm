# Запуск

Чтобы скомпилировать программу и запустить её в отладчике выполните команду:

```bash
make
```

Подробности запуска можете узнать в файле `makefile`.

# Отладка

Память:
```
(lldb) x -s4 -fb -c4 0x1fc9
```

Регистры:
```
(lldb) reg r
```

Установить бряку:
```
(lldb) b start
```

Запустить:
```
(lldb) r
```

Следующий шаг:
```
(lldb) n
```
